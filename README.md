# Spring Boot Login Application

This is a simple Login and Registration application built using Spring Boot and Java.

## Features
- User Registration
- User Login
- MVC Architecture
- Database integration using JPA & Hibernate

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Hibernate / JPA
- MySQL
- Maven

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/AishwaryaNimse/springboot-login-app.git

2. Open the project in Eclipse or IntelliJ

3. Configure MySQL database in application.properties

4. Run the project as Spring Boot Application

5. Open browser and visit:
   http://localhost:8080
